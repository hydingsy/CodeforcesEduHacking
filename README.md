# Codeforces Edu Hacking
## **简介**

Codeforces Edu 场错误代码查找器。

可在每次 Edu 结束后 24 小时内食用。



## **暂定需求**

- [x] 可选择 contest
- [x] 可指定所查询的题目（A、B、C、D、E、F?）
- [x] 可指定若干组输入数据以及正确的输出结果（使用 `---` 分隔）
- [ ] 并行化查询
- [ ] 可选择本地支持的编译环境
- [x] 最终查询
- [ ] 评判 [暂停 / 继续]
- [ ] 区间分段查询
- [x] 可查看当前 Hack 榜单





## **需要做什么**

- 本地 `GCC/G++` 环境变量
- 本地 `Java 8` 环境变量
- 本地 `Python 2/3` 环境变量

静静的看着吧！
